# Animated-Portfolio
This is a simple HTML/CSS template for a personal portfolio website.
It is a modern and visually appealing design with animations and transitions that make the website stand out.

## Technologies Used

- HTML
- CSS
- JavaScript
- Boxicons
- Typed.js
